# Release Notes - YetAnotherTraderMod v0.0.4

## Changes

* Updated mod version to `0.0.4`.
* Continued development of the custom trader **Tony Volkov**.
* Added/confirmed Tony as a fully separate trader entry.
* Added four loyalty levels for Tony.
* Added mixed trader assort across loyalty levels.
* Added purchasable weapons, weapon parts, armor, rigs, helmets, plates, meds, food, drinks, and ammo.
* Added ruble purchase options.
* Added selected barter offers.
* Added insurance support for Tony.
* Added repair support for Tony.
* Updated Tony’s trader identity, description, and back-room fixer theme.
* Continued balancing inventory placement across loyalty levels.
* Continued moving stronger gear and ammo toward later progression.
* Improved trader progression direction from basic supplies to stronger late-game equipment.
* Continued cleanup of mod structure and install layout.

## Current Content

* Custom trader: `Tony Volkov`
* Trader nickname: `Tony`
* Trader location: `A locked back room beneath Tarkov.`
* Four loyalty levels
* Cash offers
* Barter offers
* Insurance service
* Repair service
* Mixed survival, weapon, armor, ammo, and utility inventory

## Not Implemented Yet

* Full Tony questline is not implemented yet.
* Quest-based trader progression is not implemented yet.
* Quest-locked inventory unlocks are not implemented yet.
* Side quests tied to cheaper trader deals are not implemented yet.
* Barter-to-cash unlocks after quest completion are not implemented yet.
* Custom quest items are not fully implemented yet.
* Tony-themed weapon presets are still planned.
* Long-term inventory balance is still being adjusted.

## Planned Direction

Future updates will focus on:

* Tony’s full questline
* Side quests for other traders
* Quest-based unlocks
* Better barter progression
* More Tony-themed items and presets
* Expanded lore and trader messages
* Better long-term balance across all loyalty levels

## Target

* SPT `4.0.x`
* Built/tested around SPT `~4.0.11`

## Install Path

Final folder path should be:

`SPT/user/mods/YetAnotherTraderMod`

## Notes

This is still a beta release. Inventory, prices, barters, trader services, and progression are subject to change.
